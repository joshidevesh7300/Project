# Cloud Lab 

## Overview
This project automates the creation of DevOps practice labs for students
using Jenkins, Terraform, Docker, and AWS EC2.

## Workflow
1. Jenkins pipeline triggered
2. Docker-based DevOps lab image built
3. Terraform provisions AWS EC2 lab VM
4. Student receives ready-to-use lab

## Tools Used
- AWS EC2
- Terraform
- Jenkins
- Docker
- Git

## Real-World Use Case
Designed for EdTech platforms to:
- Reduce manual lab setup
- Control cloud costs
- Provide consistent DevOps learning environments
